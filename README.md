## 📖 ¿Qué es Go?

**Go** (también conocido como **Golang**) es un lenguaje de programación de código abierto creado por **Google** en 2007 y lanzado públicamente en 2009. Fue diseñado por tres ingenieros de Google: **Robert Griesemer, Rob Pike y Ken Thompson** (este último, co-creador de Unix).

### Características principales:

- **Compilado**: Se convierte directamente a código máquina, lo que lo hace extremadamente rápido
- **Tipado estático**: Detecta errores de tipos antes de ejecutar el programa
- **Concurrencia nativa**: Puede manejar múltiples tareas simultáneas de manera sencilla (goroutines)
- **Sintaxis simple y limpia**: Solo tiene 25 palabras reservadas, fácil de aprender
- **Recolección de basura**: Gestiona la memoria automáticamente
- **Multiplataforma**: Compila para Windows, macOS, Linux y más

## 🎯 ¿Para qué se utiliza Go?

Go es especialmente popular en:

### 1. **Backend y APIs**
- Servidores web y microservicios
- APIs RESTful y GraphQL
- Aplicaciones empresariales escalables

### 2. **DevOps y Cloud**
- Herramientas de línea de comandos (CLI)
- Infraestructura como código
- Contenedores y orquestación

### 3. **Aplicaciones de red**
- Servidores proxy
- Balanceadores de carga
- Sistemas distribuidos

### 4. **Herramientas del sistema**
- Scripts de automatización
- Procesamiento de datos
- Utilidades del sistema
